table ကို border လေးနဲ့ပြချင်ရင် table tag မှာ border="" attribute ကိုသုံးပါ column ပေါင်းချင်ရင် th or td tag မှာ colspan="" attribute သုံးပါ။ row ပေါင်းချင်ရင် rowspan="" attribute သုံးပါ။ မှတ်ချက်။ row များသည်အောက်ကိုသာပေါင်းသွားမှာဖြစ်ပါတယ် table တွေကို section လေးတွေခွဲပြီးသတ်မှတ်ချင်ရင်တော့ tag လေးတွေရှိပါတယ်။ team နဲ့အလုပ်လုပ်တဲ့အခါ တခြားသူအတွက်လည်းအဆင်ပြေ ကိုယ်ပြန်ပြင်စရာရှိရင်လည်းလွယ်ကူအောင်ပါ။ စနစ်လည်းကျပါတယ်။
for table header - theader
for table body - tbody
for table footer - tfoot

Division Tag များနှင့် Web-Page တစ်ခုအား ခွဲခြမ်းစိတ်ဖြာခြင်း

Division tag <div> </div>များကို section များပိုင်းခြားရန်အသုံးပြုပါတယ်။ Web-Pageတစ်ခုအား develpnment ပြုလုပ်ရာတွင်

  • header
  • section
  • footer

စသဖြင့် ပိုင်းခြား၍ ရေးမည်ဆိုပါက division များကိုစီမံခန့်ခွဲရာမှာပိုမိုလွယ်ကူမှာဖြစ်ပါတယ်။ Div tag များကိုသက်ဆိုင်ရာ အပိုင်းတစ်ပိုင်းစီတွင် ထည့်သွင်းခြင်းဖြင့် စနစ်ကျကာ ရှင်းလင်းလာမည်ဖြစ်ပါသည်။
