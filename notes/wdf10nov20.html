Anchor Link

အတွင်းမှစာသားများကိုတော့ lorem ipsum site ကနေ copy ယူပါ။ များနိုင်သလောက်များအောင် copy ပွားပါ။ဆရာက ဆယ်ခါ ပွားခိုင်းပါတယ်။ပြီးရင်စာပိုဒ်ကို ငါးခါ ထပ်ပွားပါ။ <p> tag များကို <div> တစ်ခုစီ အောက်တွင် ခွဲ၍ ထည့်ထားဖို့အရေးကြီးပါတယ်။ ပြီးရင်တော့ div tag မှာ id="" ဆိုတဲ့ attribute ကိုသုံးပြီး id လေးတွေသတ်မှတ်ပေးပါ။ id ဟာ unique ဖြစ်ရပါမယ်။ id တွေဟာတူနေလို့မရသလို ကိုယ်လွယ်လွယ်ခေါ်သုံးနိုင် မှတ်မိနိုင်မယ့် id မျိုးသတ်မှတ်ပေးရပါမယ်။ custom value မို့ ကိုယ်ဘာထားထားရသလို ထားထားတဲ့အတိုင်း ခေါ်သုံးရမှာပါ။ id ရဲ့ value ကို anchor tag မှာ link လုပ်မှာပါ။ ဒါဆို anchor လုပ်ထားတဲ့ hyper link ကို click လုပ် လုပ်တဲ့အခါ ချိတ်ထားတဲ့ id value ရှိတဲ့ div ကိုခေါ်သွားပေးမှာပါ။ html မှာ id ကို ခေါ်သုံးချင်ရင် # ခံပြီးသုံးပါတယ်။ အောက်မှာ code လေးတွေကြည့်ကြည့်ပါ။ result တော့မပြတော့ပါဘူး။ ကိုယ့်ဘာသာ try ကြည့်ပါ။

အထက်ကပုံအတိုင်း ရေးရပါတယ်။ layout ပုံစံပါ။ user friendly ဖြစ်အောင် attribute တွေရှိပါသေးတယ်။ label tag မှာ lable click တာနဲ့ auto culer ကျ နေအောင် for ဆိုတဲ့ attribute နဲ့ input tag ရဲ့ id နဲ့ချိတ်ရပါတယ်။ input tag မှာလည်း id attribute သုံးရမယ်ပေါ့။ id က ဒီထက်ပိုသုံးနိုင်ပါသေးတယ်။ CSS နဲ့ design ဆွဲချင်တဲ့အခါ (သို့) JavaScript ကတစ်ဆင့် data base နဲ့ချိတ်ဆက်တဲ့အခါ တွေမှာသုံးပါသေးတယ်။ အောက်မှာ attributeအချို့ကိုဖော်ပြပေးပါမယ်။

attribute for label tag

  • for="" (to bind the related element)

attribute for input tag

  1. type="" (types for user input)

    values

    • text
    • number
    • password
    • radio
    • checkbox
  2. name="" (to indicate database for backend PHP user)
  3. id="" (specifies a unique id )
  4. class="" (specifies one or more classnames for an element)
  5. value="" (user input value)
  6. placeholder="" (specifies a short hint to input field
