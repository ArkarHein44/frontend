Result

အခြား stand alone attributeများရှိပါသေးတယ်။ readonly ဆိုတာရှိပါတယ် input box မှာသွားပေးထားမယ်ဆိုရင် culsor ပဲချလို့ရပါမယ်။ ရိုက်ထည့်လို့မရတော့ပါဘူး။ disabled ကိုထည့်ရင်တော့ cursor ပါ ချရမှာမဟုတ်တော့ပါဘူး။ radio button တွေ check box တွေမှာ ကြိုရွေးထားချင်ရင်တော့ selected သုံးလို့မရပါဘူး။ checked ဆိုတဲ့ stand alone attribute ကိုသုံးပါ။ select option မှာ အများကြီးရွေးလို့ရစေချင်ရင် multiple ကိုသုံးပါ။ အများကြီးတစ်ခါတည်းရွေးထားတယ်ဆိုရင်တော့ array ကိုသုံးရပါတယ် php အတွက်ပါ။ backend မှာ database ထည်းကို ထည့်ဖို့ပါ arrayအတွက် [] သုံးပါတယ် javascript မှာလည်းသုံးပါတယ်

    • range အတွက် min နဲ့ max ဆိုတဲ့ attribute ကိုသုံးရပါတယ်။ step ဆိုတဲ့ attribute နဲ့ အနည်းဆုံးဘယ်လောက် ထိတိုးလို့ လျော့လို့ရမယ်ဆိုတာကို ချိန်ညှိလို့ရပါတယ်။ file ဆိုတဲ့ attribute ကိုသုံးမယ်ဆို ကိုယ်က ရွေးစေချင်တဲ့ file type ကို accept ဆိုတဲ့ attribute နဲ့ကြိုသတ်မှတ်ပေးထားနိုင်ပါတယ်။ image အတွက် image/* နဲ့ video အတွက် video/* ဆိုတဲ့ value ကိုသုံးနိုင်ပါတယ်။ image နဲ့ video တွေကို mobile device တွေအတွက် camera ကိုခေါ်သုံးနိုင်အောင် capture="user" ဆိုပြီး attribute ထည့်သုံးနိုင်ပါတယ်။ ဒါဆို camera app တွေပွင့်လာမှာပါ။ input box တွေမှာ အရင်က ဖြည့်ထားတာတွေရှိရင် auto ပြန်ပြန်ပေါ်လာတတ်ပါတယ်။ မပေါ်စေချင်ရင် autocomplete="off" ဆိုတဲ့ attribute ကိုသုံးပါ။
