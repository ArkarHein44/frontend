Image Tag <img/> (self closing tag)-Inline element

  1. External link(Online မှ ပုံများကိုယူသုံးခြင်း)
  2. Internal link(မိမိ source file မှယူသုံးခြင်း)
attributes
  1. src=""(src=source)
  2. alt=""(alternate text for an image)
  3. width=""(image width)
  4. height=""(image height)

Table Tag <table> </table> (block element)

<th> </th> means Table head
<tr> </tr> means Table row
<td> </td> means Table data (column)
table တည်ဆောက်ရာမှာ table tag ကိုသုံးပါတယ်
ပြီးရင် row တစ်ခုယူပါမယ် tr tag ကိုသုံးပါ
သူ့အောက်မှာ table head အတွက် th tag နဲ့ column လေးတွေ တည်ဆောက်ပါ
ပြီးရင်နောက်ထပ် row တစ်ခုထပ်ထည့်မယ်ဆို tr tag ပြန်သုံးပါ
row အတွက် td tag ကိုသုံးပါ table head ကိုတော့ အပေါ်ဆုံးrow တစ်ခုအတွက်ပဲသုံးပါ
