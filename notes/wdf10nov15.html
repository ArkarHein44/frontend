Image Tag <img/> (self closing tag)-Inline element

  1. External link(Online မှ ပုံများကိုယူသုံးခြင်း)
  2. Internal link(မိမိ source file မှယူသုံးခြင်း)
attributes
  1. src=""(src=source)
  2. alt=""(alternate text for an image)
  3. width=""(image width)
  4. height=""(image height)

Project တစ်ခုမှာပုံတွေကအများကြီးမို့ ပုံမှာ error တစ်စုံတစ်ရာဖြစ်ပါက ရှာရလွယ်စေရန် alt="" attribute ကိုသုံးပါ
ပုံထည့်တိုင်း alt လေးကိုထည့်သုံးတာက developer အတွက်ပိုအကျိုးရှိပါတယ်
ပုံပေါ်နေရင်တော့ alt ထည်းက value ကမပေါ်ပါ
ပုံမပေါ်ရင်တော့ alt မှာပေးထားတဲ့ value ကပေါ်နေမှာပါ
width height ချိန်းချင်ရင် atribute လေးထည့်သုံးပါ
width သို့ height တစ်ခုခုသုံးရင်ရပါတယ် ratio အလိုက် autoချိန်ပေးပါတယ်
